**Customer Satisfaction Prediction**

This project focuses on analyzing customer support ticket data to understand customer behavior, 
identify common issues, and predict customer satisfaction levels using machine learning techniques.
The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model
building, evaluation, and visualization to extract meaningful insights and improve decision-making for
customer support systems.

**Dataset**
Dataset Link :- https://drive.google.com/file/d/1Jj_fCI259KMyrt8d6gYfgrDx8WYK1E6i/view?usp=drive_link

**Problem Statement**

In modern organizations, customer support teams handle large volumes of tickets from multiple channels,
making it difficult to analyze customer behavior and service performance effectively. Without proper analysis, 
valuable insights remain hidden. This project aims to analyze support ticket data to identify trends, understand 
key factors affecting customer satisfaction, and build a machine learning model to predict satisfaction levels, 
helping improve customer experience and support efficiency.

**Methodology**

The project begins with data preprocessing to clean and prepare the dataset for analysis. Exploratory Data Analysis (EDA)
is performed to understand data distributions and patterns. Feature engineering techniques are applied to improve model 
performance by creating meaningful features. A Random Forest machine learning model is trained to predict customer satisfaction
ratings. Finally, the model is evaluated using performance metrics and visualizations to interpret results and insights.

**Conclusion**

This project demonstrates how data analysis and machine learning can be applied to customer support data to uncover patterns
and predict satisfaction levels. The analysis helps identify important factors influencing customer experience and provides
insights that can support better decision-making. The predictive model offers a data-driven approach to improving customer 
service quality and operational efficiency.
